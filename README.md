# 🚀 Django App with Azure Web App

A simple **Django web application** deployed on **Microsoft Azure Web App**, demonstrating how to build and host a Python web project in the cloud.

---

## 📌 Features

* 🔐 **User Authentication** (Login/Register)
* 📝 **Basic Web Functionality** using Django
* ☁️ **Azure Web App Deployment**
* ⚡ **Production-ready setup** with static files

---

## 🛠️ Built With

* 🐍 **Django (Python)**
* 🌐 **HTML, CSS, Bootstrap**
* ☁️ **Microsoft Azure App Service**

---

## ⚡ Description

This project shows how to develop a **Django application** and deploy it to **Azure App Service**, a cloud platform that supports Python web apps and continuous deployment from GitHub. ([Microsoft Learn][1])

It is a great starting point for learning **cloud deployment** and building scalable web applications.
